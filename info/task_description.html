Let's grow strawberries !

But . . . all kids in the hood are all fond on strawberries, we need to protect our precious plants by placing a fence.

The fence is made of 4 fence parts (a, b, c, d) of (but not necessary) different lengths. They are going to be placed around the field in a anticlockwise order.

Up to you to write a function which returns the angle (in degrees) of the inner corner between first and last fence part (a and d), so that the fence encloses the maximum area for our strawberry bed.

Thus your function is given four numbers (a, b, c, d) representing the length of the four fence parts and has to return the angle measured from the inside rounded to whole degrees for the maximum enclosed area. (fig.1)

Input: Four arguments: All integers

Output : A number, float or integer.

Acuracy: one tenth of a degree.

Example: 

    strawberryfield(100, 100, 100, 100) == 90
    strawberryfield(200, 100, 100, 100) == 60
    strawberryfield(300, 141.421356, 100, 141.421356) == 45
    strawberryfield(100, 141.421356, 300, 141.421356) == 135
    strawberryfield(300, 100, 100, 100) == 0
    strawberryfield(82, 117, 203, 123) == 119

How it is used: Surveying

Precondition:

∀ length of all sides : length of one side <= sum length of three other sides

∀ angle α of all corners : 0° <= α <= 180°
